Shows DataBase size in admin board.

== Description ==

This is a very simple plugin which just shows your DataBase size under the header of the admin control panel. Usefull for those who have database size limited by their host.
Database size appears in green if under 4 MBytes by default, else in red if over this value.


== Installation ==

 - Download db_size.zip here : http://wordpress.org/extend/plugins/db-size/
 - Unzip and copy to your wordpress plugins folder, then activate. 
 - Adjust the Alert Level to your convenience by editing line 20 in db_size.php.
   (Modify in the extensions manager)
 - That's all.
   
Enjoy ! :)
